## Install games on CFW Switch
- connect Switch to PC using USB cable (data cable!)
- open DBI on emuMMC
- Connect via mtp responer
- wait
- on PC open explorer for connected device (switch)
- copy nsp, etc files one by one in the correct order to explorer
- NS shows installation process
